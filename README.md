# 📊 Sales Performance Dashboard (Tableau)

## 📌 Project Overview
This project showcases an interactive Sales Dashboard built using Tableau to analyze business performance across multiple dimensions including sales, profit, and quantity.

The dashboard provides insights into yearly trends, product performance, and weekly variations to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze year-over-year (YoY) sales performance
- Track profit and quantity trends
- Identify best and worst performing months
- Compare product subcategories
- Highlight weekly performance deviations

---

## 🧩 Dashboard Features

### 🔹 KPI Overview
- Total Sales, Profit, Quantity
- YoY comparison with percentage change
- Visual indicators for growth/decline

### 🔹 Sales Trends
- Monthly comparison (Current Year vs Previous Year)
- Highlights highest and lowest sales months

### 🔹 Subcategory Analysis
- Sales and profit comparison across product subcategories
- Identifies profitable and loss-making segments

### 🔹 Weekly Trends
- Weekly sales and profit trends
- Average reference lines
- Above vs below average highlighting
